Etiquetas Basicas,atributos y valores.

Las etiquetas básicas de HTML se pueden definir como fragmentos de texto que se usan para dar propiedades específicas al texto cuando se escribe en código html. Tambien cuenta con los atributos de HTML que son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento.
