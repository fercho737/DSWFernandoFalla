Etiquetas Basicas,atributos y valores.


Las etiquetas básicas de HTML se pueden definir como fragmentos de texto que se usan para dar propiedades específicas al texto cuando se escribe
en código html. Tambien cuenta con los atributos de HTML que son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar
el comportamiento del elemento.

Aprendizaje de las etiquetas basicas HTML

El HTMl es el lenguaje básico para crear sitios web. en el siguiente curso vamos a hacer un sitio básicamente con este lenguaje para desarrollar un aprendizaje agradable, entendible, navegable,
accesible y adaptable a dispositivos. En el código HTML se escriben los textos que lee el usuario y se vinculan las imágenes importantes cuya función es aportar contenido. También se definirán
las estructuras, daremos jerarquía al contenido y se vincularán los archivos.
