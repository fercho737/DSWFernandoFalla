Propósito

HTML es el lenguaje muy importante cuyo empleo se redirige al desarrollo de páginas de internet. Este lenguaje está compuesto por una seríe de etiquetas que el navegador interpreta y da forma en la pantalla. HTML dispone de etiquetas para imágenes, hipervínculos que nos permiten dirigirnos a otras páginas, saltos de línea, listas, tablas, etc.
